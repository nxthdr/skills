# Skills

A collection of [Claude Code skills](https://docs.anthropic.com/en/docs/claude-code/skills) for working with the [nxthdr](https://nxthdr.dev) platform.

## What are skills?

Skills are reusable instructions that teach Claude Code how to perform specific tasks. They can be loaded into your Claude Code sessions to give the AI assistant deep knowledge about nxthdr workflows.

## Available Skills

| Skill | Description |
|-------|-------------|
| [Probing Measurements](skills/probing-measurements.md) | Run active Internet measurements using the nxthdr CLI and analyze results from ClickHouse |

## Usage

Copy the skill file you need into your project's `.claude/skills/` directory, or reference it directly in your conversation with Claude Code.

## Contributing

Have a workflow you'd like to share? Open a PR with a new skill file in the `skills/` directory.
